# Jualan Tiket 🎟️

Aplikasi berbasis Laravel untuk mengelola penjualan tiket acara secara online.

## 🚀 Fitur Utama
- Manajemen data event dan tiket
- Pemesanan dan pembayaran online
- Dashboard admin dan laporan penjualan

## ⚙️ Teknologi
- Laravel 10
- MySQL
- Bootstrap

## 📦 Cara Menjalankan
```bash
composer install
cp .env.example .env
php artisan key:generate
php artisan serve
